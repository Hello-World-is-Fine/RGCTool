CSV File Renamer 使用說明

Step 1

先用excel準備要改的filename，如下圖樣(只可以有Column A, B; Rows 數量不限)

然後另存為.csv format

Step 2

點擊按扭 "選擇檔案"，點選剛才的csv file

Step 3

點擊按扭 "Process CSV"，有一個 "rename_files.bat" file自動下載

Step 4

把剛才的 "rename_files.bat" file 放到要改名的檔案的文件夾內
